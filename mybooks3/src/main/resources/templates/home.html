Purpose

Here you can maintain own interesting thoughts, quotations, comments and notes you found while reading a book.

This is for everyday or occasional readers, people who likes reading and want to record interesting and useful thoughts.

Overview

It is responsive web application to be able from any device to write down notes as quick as possible. It is consisted of next parts:
  • Authentication â€“ registration, logging into the portal and logout. In case of forgetting password it supports generation a new password which will be send to the mail.
  • Maintain contents - create, view, update, delete and search contents about the book.
  • Profile - reader can view and update his own profile data.
  • Administrate readers - Admin type of users can maintain users' profiles by being able to update and delete their profile data.



User Types

  • Readers - people that have registered and can maintain their data about read books.
  • Admins - apart of functionalities which has common reader it has ability to administrate other users.
